Visualisation des ventes cinéma

Instructions

  1. Se connecter à l'outil d'extraction Digitick : http://admin.digitick.com
  2. Choisir la période d'export (date de début et date de fin) et cliquer sur "+"
  3. Cliquer sur "Ajouter la requête"
  4. Sélectionner au minimum les champs :
    • Transaction > Canal de vente > ID
    • Place > Produit > Nom
    • Place > Représentation > Début
    • Place > Lieu > ID
    • Place > Tarif > ID
  5. Cliquer sur "Effectuer l'export"
  6. Cliquer sur l'icône "TXT" pour enregistrer le résultat de l'export (fichier .csv) sur votre disque dur
  7. Glisser/déposer le fichier dans la zone ci-dessous
Déposer ou sélectionner le fichier .csv
